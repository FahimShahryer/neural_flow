A neuron takes a few numbers in, does a tiny bit of math, and sends one number out. Every neural network you'll ever see is built from this one idea, repeated thousands of times.

Instead of reading about it — play with one.

<NeuronPlayground />

**Drag the weight sliders.** Each weight controls how much one input matters. Set a weight to **0** and that input stops affecting the output entirely. Make a weight **negative** and that input now pushes the output down.

**Try each activation.** `linear` just passes the sum through. `relu` cuts off anything negative. `sigmoid` squashes the output between 0 and 1. `tanh` squashes between −1 and +1. You'll feel why each one exists before the next lesson explains it.

## The math, in one line

$$
y = f\!\left(\sum_{i} w_i x_i \;+\; b\right)
$$

That's it. Everything else in this module is built from this:

- $x_i$ — an **input** (you just dragged these)
- $w_i$ — a **weight** (you just dragged these too — they decide how much each input matters)
- $b$ — a **bias** (a constant the neuron adds no matter what)
- $f$ — an **activation function** (the switch at the top of the playground)

In the next lesson you'll stack many of these neurons into a **layer** and see why that unlocks everything a network can do.
